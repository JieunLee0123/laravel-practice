## Installation
### lievewire3<br/>
1. install livewire3<br/>
composer require livewire/livewire:^3.0@beta<br/>

### volt<br/>
1. install volt<br/>
composer require livewire/volt:^1.0@beta<br/>
2. install Volt's service provider file<br/>
php artisan volt:install<br/>
